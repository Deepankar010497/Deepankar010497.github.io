Professional Project Portfolio

Real-world systems I’ve delivered with AI, ML, LLMs, and scalable microservices

Multi-Agent LLM System for SDLC Intelligence

Designed and deployed a planner–orchestrator–responder architecture using LangGraph and AutoGen to extract insights from structured and unstructured SDLC data. Agents called APIs as tools, executed based on personas, and generated payloads dynamically across workflows.

Integrated with UI for tracking agent decisions, connected to Knowledge source of structured and unstructured data for end-to-end visibility, and supported payload chaining with dynamic API orchestration using Function calling Agents. This system became a core module in the product suite and helped land enterprise demo deals. Appreciated and implemented by various internal teams within the organization.

Tech Stack: LangGraph, AutoGen, OpenAI, Python, Docker, Azure AKS, FastAPI, Elasticsearch, ChromaDB, ArangoDB, Gradio

Hybrid RAG System with Graph + Vector Retrieval

Developed and deployed a hybrid Retrieval-Augmented Generation (RAG) system combining vector-based search with graph-based chunk traversal. Enabled high-recall contextual LLM responses across structured, semi-structured, and long-form documents.

This pipeline supports both enterprise document QA and agent integration workflows. Optimized for modular plugin design and response streaming. Seamlessly scaled using ChromaDB and Elasticsearch for retrieval layers.

Tech Stack: LangChain, ChromaDB, FAISS, Elasticsearch, LLMs, Python, ArangoDB, Semantic Caching

Predictive Infra Monitoring & Anomaly Detection

Engineered forecasting workflows for infrastructure metrics using NeuralProphet and LightGBM to anticipate system load, memory spikes, and CPU trends across production systems.

Integrated anomaly detection with real-time RCA explanation generation using LLM-based prompt workflows and log analysis. Built dashboards and API-based alert triggers for proactive resource scaling and VM sizing in production.

Tech Stack: NeuralProphet, LightGBM, Python, LLM APIs, Log Analytics, Azure AKS, Docker

Document & Image to Text Intelligence

Built an end-to-end pipeline combining OCR (Tesseract), image classifiers, captioners, and semantic chunking techniques to extract structured text from scanned documents and images.

Integrated RAG-based querying over the extracted data using LLMs and multimodal inference. Designed as a plug-and-play module to support downstream agent workflows and enhanced with a Gradio UI for interactive visual demos.

Tech Stack: Tesseract OCR, OpenCV, LLMs, Captioning, Gradio, Hybrid RAG, Python

Regression Test Case Optimization System

Created a content-based recommender system to suggest relevant test cases from large test libraries using semantic text similarity — drastically reducing time for regression testing in microservice-based architecture.

Engineered using TF-IDF, Word2Vec embeddings, LDA/NMF topic modeling for dynamic clustering. Enabled QA teams to select critical test cases across modules and improved cycle coverage with lower redundancy.

Tech Stack: TF-IDF, Word2Vec, LDA, NMF, Python, NLP Feature Engineering, Flask
